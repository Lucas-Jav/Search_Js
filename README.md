## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Lucas-Jav/Search_Js.git
```

Entre no diretório do projeto

```bash
  cd Search_Js
```

Instale as dependências

```bash
  npm install
```




## Inicialização


#### Start Web Scraping
```powershell
  npm start
```
